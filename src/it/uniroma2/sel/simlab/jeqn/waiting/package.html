Provides jEQN components to simulate User Waiting Systems.

It is futher subdivided into {@link policies}, {@link storages} and {@link times}.

There are two kind of Waiting Systems made available: PreemptiveWaitingSyste and NonPreemptiveWaitingSystem.

The former simulates a user waiting system that allows incoming user to preempt user under process if certain conditions are met.

The latter simulates a user waiting system that acts passively compared to the preemptive one as it waits for next user request to forward a new user to the in cascade jEQN entity.

Each type must be properly connected to a ServiceCenter which behaviour supports to the dynamic of the specific type.

Each type can be built up by setting its properties
